# Registro de Usuario — U4
Formulario de registro con validación completa del lado del cliente usando JavaScript y Constraint Validation API.

## Tecnologías
- HTML
- CSS
- JavaScript ES6

## Funcionalidades
- Validación de campos obligatorios
- Validación de correo electrónico
- Validación de contraseña con regex
- Confirmación de contraseña
- Validación de teléfono
- Mensajes de error por campo
- Mensaje de registro exitoso

## Ejecución
Abrir index.html con Live Server en Visual Studio Code.
